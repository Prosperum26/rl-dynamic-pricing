# Scripts

CLI utilities (not imported as library code).

| Script | Command |
|--------|---------|
| Preprocess category-level (RL) | `python -m scripts.preprocess_ecommerce` |
| Preprocess + product tables | `python -m scripts.preprocess_ecommerce --with-products` |
| Product-level only | `python -m scripts.preprocess_products` |
| Generate synthetic data | `python -m scripts.generate_data` |
| Environment smoke test | `python scripts/quickstart.py` |
